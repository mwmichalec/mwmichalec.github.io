DYSSEGARD KARATEKLUB

A basic, responsive website created as a group project for Dyssegard Karateklub located in Copenhagen. Gives a great user experience based on 960 pixel width grid. Scales down on smaller devices, adjusting to smaller laptop, tablet and smartphone sizes, keeping the website consistent and user friendly. The website has been tested using usability tests and received very positive feedback. Website was completely coded by me, using HTML5, CSS, simple Java Script and is based on Bootstrap's responsiveness. It was coded based on original design, which was slightly tweaked and appreciated by the group I worked with.

The website is available for viewing upon request.

ONLINE PORTFOLIO

Website created from the bototm by myself and fo myself. It shows knowledge of design, usability, HTML5, CSS, basic Java Script, jQuery libraries, Bootstrap. Website is completely responsive, making it's elements scale down to smaller devices and larger devides, originally up to FHD resolution. Website contains "Hamburger menu", and it's main purpose is fast and quick navigation on desktop, but was aimed mostly towards mobile devices, so user can quickly switch before different sections of the website. It also includes home button allowing to simply go back to the front page and navigate from there on. Website is kept in simple, flat design, which itself became really popular due to it's various benefits.

The website is available here: http://bamsesl.github.io/
